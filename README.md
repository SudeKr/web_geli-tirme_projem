# Web Geliştirme Projesi - Film Arşivi

Bu projeyi Web Geliştirme projesi için React kullanarak hazırladım. Temel olarak bir film izleme listesi (Watchlist) uygulamasıdır.

## Neler Kullandım?
* ReactJS 
* Tailwind CSS 
* LocalStorage 

## Özellikler
1. Film ekleme 
2. Filmler listeleme
3. İzlenen filmleri güncelleme
4. Film silme
   

## Klasör Yapısı
Components, Pages ve Interfaces klasörlerini oluşturdum. Proje ana dosyası index.html içinde duruyor.
